# Spectacular Wesley: Car Photography Portfolio 

Welcome to My Photoghrapy web! This is a simple, responsive photography portfolio website dedicated to showcasing automotive photography.

##  About the Project

As an automotive enthusiast, I wanted to create a clean, modern space to display my car photography. This website highlights various car makes and models, focusing on capturing details

## Built With

- **HTML5** - For structuring the webpage content.
- **CSS3** - For styling, layouts, and making the site mobile-friendly.

##  Features

- **Image Gallery:** A responsive grid layout to easily view high-resolution car photos.
- **Responsive Design:** Works perfectly across desktop, tablet, and mobile phone screens.
